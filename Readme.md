# PlayPoints

**PlayPoints** es una aplicación enfocada en ayudar a personas con problemas de adicción a las apuestas deportivas a reducir y eventualmente eliminar este hábito.

A diferencia de otras apps, PlayPoints no fomenta el juego, sino que introduce mecanismos de control como **cooldowns (periodos de espera obligatorios)** entre apuestas. Estos tiempos aumentan progresivamente según el uso, dificultando las decisiones impulsivas y promoviendo un comportamiento más consciente.

## ⚙️ ¿Cómo funciona?

- El usuario registra su actividad de apuestas
- Se aplican tiempos de espera automáticos entre apuestas
- Se bloquea temporalmente la posibilidad de apostar si se detecta uso excesivo

## 🎯 Objetivo

El objetivo principal es **reducir la dependencia** de las apuestas mediante:
- Control del impulso
- Conciencia del comportamiento
- Limitaciones progresivas

> “No se trata de prohibir, sino de ayudarte a recuperar el control.”
